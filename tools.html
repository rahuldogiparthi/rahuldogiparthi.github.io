Software and Pipelines

Tools & Technologies

We develop and maintain open-source computational tools, reproducible analysis workflows, and curated datasets that support the broader genomics community.

Lab-Developed Software

Our Open-Source Tools

All tools are freely available on GitHub under open licenses. We encourage community use, issue reporting, and contributions.

🏹

HewittPipe

v2.0.1 — Python

An end-to-end single-cell RNA-seq analysis pipeline covering QC, ambient RNA correction, doublet removal, dimensionality reduction, clustering, and cell type annotation. Designed for cardiac and pulmonary datasets.

GitHub Docs
📊

CardioAtlas

v1.3.0 — R / Python

A reference atlas of human cardiac cell types derived from multi-donor snRNA-seq profiling. Includes label transfer functions, marker gene databases, and integration with Seurat and Scanpy.

GitHub Dataset
🧬

LungGraph

v0.9.2 — Python

A graph-based tool for cell-cell communication inference in spatial transcriptomic datasets from lung tissue. Supports Visium and Xenium platforms with ligand-receptor interaction scoring.

GitHub Preprint
🎖

MultiOmicsInt

v1.1.0 — R

A modular framework for multi-omics data integration combining scRNA-seq, scATAC-seq, and proteomics via weighted nearest-neighbor and sparse canonical correlation approaches.

GitHub Docs
🔍

PeakAnnotatoR

v1.0.3 — R

An R package for annotating ATAC-seq peaks with regulatory element databases, motif enrichment analysis, and linking accessible regions to putative target genes via ABC models.

GitHub Vignette
🌠

TrajectoryViz

v0.6.0 — Python

Visualization utilities for pseudotime trajectory analysis, including gene dynamics along trajectories, branching point analysis, and interactive HTML output for publication-quality figures.

GitHub Gallery
Standard Workflow

Single-Cell Analysis Pipeline

Our reproducible analysis workflow for cardiac and pulmonary single-cell datasets, from raw sequencing to biological interpretation.

Step 01

Raw Data QC

FastQC, STARsolo, Cell Ranger alignment and barcode filtering.

Step 02

Ambient RNA + Doublets

SoupX ambient correction; Scrublet and DoubletFinder removal.

Step 03

Normalization

Scran pooling-based normalization; log1p transformation; HVG selection.

Step 04

Dimensionality Reduction

PCA, UMAP, and batch integration via Harmony or scVI.

Step 05

Clustering and Annotation

Leiden clustering; CardioAtlas label transfer; manual curation.

Step 06

Downstream Analysis

DEG testing, trajectory inference, regulon analysis, and CCC.
